# Заголовки

- На странице **должен** быть `<h1>`, который описывает содержимое страницы.
- Уровни заголовков **должны** идти последовательно от большего к меньшему без пропусков.

## Правильно

```html
<h1>Каталог</h1>
<section>
  <h2>Товары</h2>
  <article>
    <h3>Товар</h3>
  </article>
</section>
<section>
  <h2>Доставка</h2>
</section>
```

## Неправильно

```html
<h1>Каталог</h1>
<section>
  <h5>Товары</h5>
  <article>
    <h1>Товар</h1>
  </article>
</section>
<section>
  <h1>Доставка</h1>
</section>
```
